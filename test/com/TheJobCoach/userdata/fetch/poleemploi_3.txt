•	Cet écran vous présente la description de l’offre d’emploi que vous avez sélectionnée.<br />
•	Si elle vous intéresse, vous pouvez y répondre selon les indications et modalités décrites au bas de l’offre. Vous pouvez également l’imprimer, l’enregistrer (en l’ajoutant à votre classeur) ou l’envoyer à un(e) ami(e).</div></div><hr/><div id="Comment_utiliser_cet_écran__"><h3 class="parallel-block-4-1"><span class="primary">Comment utiliser cet écran ?</span><span class="secondary"><a href="#top-page" class="top-link">Haut de page</a></span></h3><div class="check-info-content"><a href="#" target="_blank"></a>









				
	













•	Pour accéder à l’offre située juste avant dans la liste, cliquez sur le lien <strong>Offre précédente</strong>.<br />
•	Pour accéder à l’offre située juste après dans la liste, cliquez sur le lien <strong>Offre suivante</strong>.<br />
•	Sur certaines offres, vous pouvez cliquer sur le lien <strong>Compétences spécifiques</strong> qui vous permet de consulter les compétences spécifiques demandées par l’employeur pour le poste proposé.<br />
<br />
•	Pour répondre à l’offre, suivez les indications et les modalités précisées au bas de l’offre : les coordonnées et le moyen de contact y sont précisés. <br />
<br />
Sur certaines offres, un bouton <strong>Postuler </strong>vous permet de « télécandidater » en ligne à l’offre d’emploi (envoi automatique d’un formulaire au conseiller Pôle Emploi). <br />
Certaines offres sont proposées par nos partenaires, pour y postuler il vous suffit de cliquer sur le logo du partenaire proposant l’offre. Vous serez alors redirigé vers son site internet dans lequel vous pourrez postuler à l’offre en question.<br />
<br />
•	Si vous souhaitez conserver une offre d’emploi dans votre classeur d’offres (pour la consulter ultérieurement), cliquez sur le bouton <strong>Ajouter à mon classeur</strong> (une fois l’offre enregistrée dans votre classeur, le bouton devient non cliquable). <br />
<strong>A noter </strong>: l’ajout de l’offre dans votre classeur nécessite de créer votre espace personnel ou de vous identifier si vous en avez déjà un. <br />
<br />
•	Si vous souhaitez envoyer cette offre par e-mail à une personne de votre connaissance (dont vous connaissez l’adresse e-mail), cliquez sur le bouton Envoyer à un ami. <br />
<br />
•	Si vous souhaitez réutiliser les critères définis dans cette recherche pour éviter de les ressaisir à chaque nouvelle consultation ou afin de recevoir les offres d’emploi automatiquement par e-mail (abonnement aux offres), cliquez sur le bouton Enregistrer les critères(une fois les critères enregistrés, le bouton devient non cliquable). <br />
<strong>A noter </strong>: l’enregistrement des critères de recherche nécessite de créer votre espace personnel ou de vous identifier si vous en avez déjà un. <br />
<br />
•	Les critères de votre recherche vous sont rappelés en cliquant sur<strong> </strong><strong>Afficher les critères</strong> : si vous souhaitez les modifier, cliquez sur le bouton <strong>Modifier les critères</strong>. <br />
<br />
•	Pour démarrer une nouvelle recherche avec d’autres critères que ceux en cours, cliquez sur le bouton Nouvelle recherche.<br />
<br />
•	Pour imprimer la page, cliquez sur le bouton <strong>version imprimable</strong> en haut de page.<br />
<br />
•	Pour revenir à la page précédente, cliquez sur le bouton <strong>Retour aux résultats</strong>.</div></div><!--
